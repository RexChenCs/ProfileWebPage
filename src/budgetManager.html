Project Description

This website allows users to register and login for their own budget control. User can modify Income Source, Expense Source for analysis by long term trends or source type. User can search Income/Expense in the table by source type, date, amount, or other information in the records. Also, export existing records in the table in many various forms, such Excel, CVS, PDF, ect.
